Personal homelab to learn and experiment with IT infrastructure

1. Raspberry Pi 4 4GB
  - Pihole
  - Portainer
  - Docker
  - Home Assistant
2. Dell Optiplex #1 (AI test server)
  - Openclaw
  - Ollama (Gemma 4)
  - Proxmox
  - Kubernetes
3. Dell Optiplex #2 (Development server)
4. Netgear Managed Switch (5 port)
  - Vlans for homelab
5. Verizon Fios Router
